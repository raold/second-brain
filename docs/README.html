üìä Architecture

Second Brain uses a modern, scalable architecture:

  • Backend: FastAPI for high-performance async REST API
  • Database: PostgreSQL 16 with pgvector for vector similarity search
  • AI: OpenAI embeddings (text-embedding-3-small) for semantic understanding
  • Frontend: Responsive web interface with real-time WebSocket updates
  • Search: Hybrid search combining vector similarity and full-text search

üîß API Endpoints

Core endpoints for memory management:

  • POST /api/v2/memories - Create a new memory
  • GET /api/v2/memories - List all memories
  • GET /api/v2/memories/{id} - Get specific memory
  • PUT /api/v2/memories/{id} - Update memory
  • DELETE /api/v2/memories/{id} - Delete memory
  • GET /api/v2/memories/search - Semantic search
