⚡ Quick Start
Example API calls
# Generate a weekly report curl -X POST http://localhost:8000/synthesis/reports/generate \ -H "Authorization: Bearer demo-token" \ -H "Content-Type: application/json" \ -d '{ "report_type": "weekly", "format": "pdf", "include_visualizations": true }' # Schedule a memory for spaced repetition curl -X POST http://localhost:8000/synthesis/repetition/schedule/{memory_id} \ -H "Authorization: Bearer demo-token" \ -d '{"algorithm": "sm2"}' # Connect to WebSocket for real-time updates const ws = new WebSocket('ws://localhost:8000/ws/connect?token=demo-token'); ws.onmessage = (event) => { const data = JSON.parse(event.data); console.log('Event:', data.type, data.data); };
