## 🛠️ **Installation & Research Setup**

### **Quick Start for Researchers**
```bash
# Clone the research platform
git clone https://github.com/yourusername/second-brain.git
cd second-brain

# Setup research environment  
python -m venv research_env
source research_env/bin/activate  # Windows: research_env\Scripts\activate
pip install -r requirements.txt

# Initialize research infrastructure
python -m app.research.setup_research_env
python -m app.cognitive.setup_benchmarks

# Start research-enabled system
docker-compose -f docker-compose.research.yml up -d
python -m app.main --research-mode
```

### **Cognitive Benchmark Validation**
```bash
# Run human memory correlation tests
python -m app.cognitive.benchmarks.ebbinghaus --validate
python -m app.cognitive.benchmarks.von_restorff --test-distinctiveness  
python -m app.cognitive.benchmarks.serial_position --curve-analysis

# Energy efficiency profiling
python -m app.research.energy_profiler --component=all --benchmark=true
```
